Settings
Configure your Anthropic API key and GitHub credentials. All keys are session-only — never stored on disk.
🤖 Anthropic API Key
Required to generate summaries. Get yours at console.anthropic.com → API Keys → Create Key.

🔒 Security: Stored in sessionStorage only. Clears automatically when you close this tab. You re-enter it each session. Never written to localStorage, disk, or any server.
🔑 GitHub Personal Access Token
Required only if you want to push summaries directly to GitHub.
Go to github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token.
Under Repository permissions set Contents → Read and write. Select only your papers repo.

🔒 Security: Also stored in sessionStorage only.
📁 GitHub Repository
Which repo to push summaries to. These non-sensitive values are remembered across sessions.
📂 File Path — You Choose Every Push
When you push a summary, a modal lets you edit the file path and commit message before anything is sent. The path is pre-suggested as {domain}/{slug}.md — e.g. supply-chain/lean-inventory-paper.md — but you can type any path you want.
🐙 Push to GitHub
Review and edit the path and commit message. Nothing is pushed until you confirm.
